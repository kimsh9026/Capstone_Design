# Capstone Design Idea
====    
### 1일 1 Idea 필수 
----
### 1. PC방, 당구장, 볼링장 등 여가 시설 빈 자리 조회 서비스
<li> 빈 자리 조회 기능
<li> 빈 자리 예측 기능 (몇분 후 사용 종료 등)
<li> 예약 기능
<li> 결제 기능 </li>

### 2. 우할관 (우리 할머니 관절)
+ 날씨 안내 어플
+ 원하는 지역, 시간 + 조건 설정
+ EX) 포항 양덕, 아침 8시 + 강수 확률 60% 이상
+ 조건 충족시에만 알림
+ 유사 어플 : 비가 올까요, 비 옴 알림(엄청 옛날 어플)

### 3. 그게 좋아
+ 음식 추천 어플
+ 어디가 아플 때, 잠 안올 때, 등등 카테고리별로 좋은 음식, 차, 음료 등을 안내
+ 데이터 입력 노가다 필요
+ 유사 어플 : 힌트
